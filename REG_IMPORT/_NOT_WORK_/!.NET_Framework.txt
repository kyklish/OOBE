Always use the last CLR version installed. This way the user does not need to have older versions of [.NetFx] installed in the system to run older applications targeting those older [.Net Framework] versions like the always annoying [.NetFx 3.5].

reg add "HKLM\SOFTWARE\Microsoft\.NETFramework" /v "OnlyUseLatestCLR" /t REG_DWORD /d "0x1" /f
reg add "HKLM\SOFTWARE\Wow6432Node\Microsoft\.NETFramework" /v "OnlyUseLatestCLR" /t REG_DWORD /d "0x1" /f
